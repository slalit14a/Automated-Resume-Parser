# 📄 Automated Resume Parser

## 🚀 Overview

Automated Resume Parser is a Python-based web application that helps recruiters and organizations automatically extract important information from resumes. The system analyzes uploaded PDF resumes and extracts details such as candidate name, email address, phone number, skills, education, and work experience.

The project is designed to reduce manual resume screening efforts and improve the efficiency of the recruitment process.

---

## ✨ Features

* 📄 Upload PDF resumes
* 🔍 Automatic text extraction
* 👤 Extract candidate name
* 📧 Extract email address
* 📱 Extract phone number
* 🛠️ Extract technical skills
* 🎓 Extract educational qualifications
* 💼 Extract work experience
* 🗄️ Store parsed data in database
* 🔎 Search candidate records
* 🎨 Modern and updated UI design
* ⚡ Fast and user-friendly interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Automated-Resume-Parser.git
```

### Move to Project Folder

```bash
cd Automated-Resume-Parser
```

### Install Dependencies

```bash
pip install flask pdfplumber
```

### Run Application

```bash
python app.py
```
